# CardsGame

A fun card game web app, created by AI using Vue in four hours.

## Project Debug

### Install dependent libraries

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

## Publish to GitHub

### Install packages

```sh

npm install gh-pages --save-dev

```

### Run

```sh

npm run build
npm run deploy

```
